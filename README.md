# 📝 To-Do List Application
A simple, modern, and responsive **To-Do List Application** developed during my **Full Stack Development Internship at Nexsoft Solutions**. This application helps users organize and manage their daily tasks efficiently through an intuitive and user-friendly interface.

## 📖 Overview
The To-Do List Application allows users to add, complete, and remove tasks, making daily task management simple and productive. The application features a clean design, responsive layout, and smooth user interactions.

## ✨ Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📱 Fully Responsive Design
- 🎨 Clean and Modern User Interface
- ⚡ Fast and Lightweight
- 🌐 Cross-Browser Compatible
- 💻 User-Friendly Experience

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
